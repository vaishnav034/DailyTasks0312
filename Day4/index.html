Visual Studio Code

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control.It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs,such as Visual Studio IDE.

Features

Supports a variety of programming languages, including JavaScript, TypeScript, and Node.js. It also includes features like debugging, syntax highlighting, code completion, and version control

Ease of use

VS Code is easy to install and use, and it doesn't take up too much space. It's a good choice for beginners, and it's also useful for advanced programmers.

Here are some other things you can do with VS Code:

  • Run tasks
  • Use snippets
  • Refactor code
  • Use embedded version control with Git
  • Select multiple files and run actions on them
  • Use the Compare Selected command to diff two files

Vs code Extensions

S.Number Extensions
1

Live Server

    Purpose: Automatically reloads the browser when changes are made to files in a web project.
    Use Case: Launches a local development server with live reload functionality, making it easier to see the effects of changes in real-time.
    Why Useful: Saves time by eliminating the need to manually refresh the browser after every change.
2

Live preview

    Purpose: Offers a live preview of your HTML and CSS directly within VS Code.
    Use Case: Displays a real-time preview of your web project inside the editor, minimizing the need to switch to a browser.
    Why Useful: Helps debug layout issues and preview designs quickly without leaving the coding environment.
3

Emmet Live

    Purpose: Speeds up HTML and CSS coding by using abbreviations that expand into full code blocks
    Use Case: Instead of typing full tags or styles, you use shortcuts like div.container>ul>li*5 to generate structured code
    Why Useful: Significantly reduces repetitive typing and accelerates front-end development.
4

Prettier

    Purpose: Automatically formats your code according to a consistent style guide
    Use Case: Formats JavaScript, CSS, HTML, and other code on save or command.
    Why Useful: Ensures clean, readable, and consistently formatted code, especially in team projects.
5

Material Theme

    Purpose: Provides a visually appealing UI theme for VS Code.
    Use Case: Changes the editor's appearance to a modern, user-friendly design.
    Why Useful: Enhances readability and reduces eye strain with clean and aesthetic visuals.

Emments

S.Number Emments Details
1 ! Use: Quickly generates the basic structure of an HTML document.
Example:
Press: Tab or Enter
2 > - Child Element Use: Creates a child element inside a parent element
Example:
Abbreviation: div>ul>li
3 + - Sibling Element Use: Creates sibling elements at the same level.
Example:
Abbreviation: header+main+footer
4 ^ - Move Up (Parent Element) Use: Moves up one level in the hierarchy to create a sibling for the parent element
Example:
Abbreviation: div>ul>li^p
5 {} - Add Text Content Use: Adds text content inside an element.
Example:
Abbreviation: p{Hello, World!}
6 HTML Code Abbreviations Use: Emmet lets you write short abbreviations that expand into valid HTML structures.
Example:
Abbreviation: div.container>ul>li*3
7 Customizing Classes and IDs Use: You can include class or id attributes directly in the abbreviation.
Example:
Abbreviation: div#main.container
